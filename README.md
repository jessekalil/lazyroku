## Elevator Pitch

Managing Roku devices from the command line shouldn't require juggling multiple tools or memorizing obscure commands. Whether you're debugging remote control issues, monitoring device status, or automating deployment workflows, you deserve a streamlined interface that respects your time and terminal-native workflow.

If you're tired of switching between web interfaces, mobile apps, and scattered command-line scripts to do basic Roku device management, lazyroku might be for you.

## Features

- **Device Discovery**: Automatically detect and list Roku devices on your network
- **Remote Control**: Send button presses and navigate your Roku from the terminal
- **App Management**: Launch, install, and manage channels and applications
- **Status Monitoring**: Real-time device status and connection information
- **Keyboard-Driven**: Full keyboard navigation with intuitive keybindings
